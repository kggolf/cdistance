Features. Possibilities of this App.
  • Markers can be dragged (from the map) to define a new location (Current location and Destination).
  • You can use the input field to get an autocomplete dropdown from google (current location and destination).
  • Current position can be defined or changed by dragging the marker or typing in the field.
  • Last destinations selected can be shown and re loaded.
  • Infowindow (in the map) can be shown when clicking the markers.
